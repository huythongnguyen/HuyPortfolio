# Đề án dịch thuật

Đề án này hướng đến việc dịch lại những văn bản quan trọng để chia sẻ cùng cộng đồng. Hy vọng có thể giúp những tri thức này lan tỏa rộng hơn, vượt qua các rào cản ngôn ngữ.

# Translation Project

This project aims to translate significant texts and share them with the community. Hope to make this knowledge accessible to a wider audience, transcending language barriers.
